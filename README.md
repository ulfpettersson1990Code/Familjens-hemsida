# Familjens-hemsida
Hemma projekt för familj.


Detta projekt är ett resultat av självstudier inom modern webbutveckling. Under byggets gång har jag aktivt använt:
Officiell dokumentation: Firebase och SMHI API för att förstå datastrukturer och autentisering.

AI som tekniskt bollplank: För att diskutera arkitekturval, felsöka logik och implementera best practices kring ES6-moduler och säkerhet (t.ex. hantering av miljövariabler).
