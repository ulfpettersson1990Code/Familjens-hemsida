# Familjens-hemsida
Hemma projekt för familj.


Detta projekt är ett resultat av självstudier inom modern webbutveckling. Under byggets gång har jag aktivt använt:
Officiell dokumentation: Firebase och SMHI API för att förstå datastrukturer och autentisering.

AI som tekniskt bollplank: För att diskutera arkitekturval, felsöka logik och implementera best practices kring ES6-moduler och säkerhet (t.ex. hantering av miljövariabler).

Standard appen hade inte timer som funktion så har fått bygga den funktionen för att kunna köra en mer ekonomisk struktur på min elförbrukning.
problemet som stöttes på var timeout på integrationen mellan mjuk och hårdvaran vilket resulterade i den lösning som nu finns i bridge.js

försökte även att testa min sida tex genom att bryta mig förbi authentiseringen vilket gjorde att jag hittade en svaghet som nu är fixad.
